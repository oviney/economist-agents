    Lessons Learned¶

    1. Mock Patching: Must patch at import location, not usage location

    2. @patch('llm_client.call_llm') not @patch('agents.graphics_agent.call_llm')

    3. Mock Configuration: Complex mocks need proper setup

    4. mock_metrics_collector.current_session = {'charts': []} prevents subscriptable errors

    5. Keyword Arguments: Check kwargs when positional args fail

    6. calls[0].kwargs['success'] instead of calls[0][0][1]

    7. Test Fixtures: Reusable fixtures improve test clarity

    8. mock_llm_client, valid_chart_spec, temp_output_path, etc.
