    Prevention Mechanisms: 1. Writer prompt explicitly warns about missing layout 2. Agent reviewer checks YAML for layout: field 3. Publication validator blocks articles without layout

    Prevention Mechanisms: 1. Writer prompt shows EXACT chart embedding syntax required 2. Agent reviewer checks for chart markdown if chart exists 3. Publication validator scans for chart references

    Issue #17: Duplicate Chart Display¶

    Confidence: ✅ 95% (high - dual check)
